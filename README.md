# Como rodar o projeto:

**Certifique-se de ter o ghci instalado no seu ambiente de desenvolvimento**

1. Digite o comando abaixo:

```
cabal clean
```

2. Digite o comando abaixo:

```
cabal build
```

3. Digite o comando abaixo para rodar o projeto:

```
cabal run
```
